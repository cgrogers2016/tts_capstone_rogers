## SASS
- Run Scout app to compile SASS  
### Variables
`$variablename: lightblue;`
### Nesting
```SCSS
main{
    h1{
      color: blue;
    }
}
```
### Color Operators  
```SCSS  
h1{color: lighten(blue, 20%)}
h1{color: darken(blue, 20%)}
```
